# Simulated data on attitudes towards institutions

Simulated data to use for examples and function testing.

## Usage

``` r
trust
```

## Format

### `trust`

A data frame with 1 000 rows and 12 columns:

- police:

  Generally speaking, can the following be trusted: Police

- government:

  Generally speaking, can the following be trusted: Czech government

- eu:

  Generally speaking, can the following be trusted: European Union

- army:

  Generally speaking, can the following be trusted: Army

- scientists:

  Generally speaking, can the following be trusted: Scientists

- eu_index:

  Attitudes towards EU index

- nat_index:

  Openness towards society index

- age:

  Age

- W:

  Sampling weight

- biggest_concern1:

  Biggest concern about society in the last month

- biggest_concern2:

  Second biggest concern about society in the last month

- biggest_concern3:

  Third biggest concern about society in the last month
